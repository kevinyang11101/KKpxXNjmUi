# Vue3纯前端后台管理系统 java2

## 项目概述

本项目是基于Vue3的企业级后台管理系统，采用纯前端技术构建，无数据库依赖。系统具备完整的权限管理、数据管理、系统设置等功能，适用于中高级前端开发者学习和企业级项目的二次开发。

## 技术栈

- **前端框架：** Vue3
- **组件库：** Element Plus
- **构建工具：** Vite
- **状态管理：** Pinia
- **路由管理：** Vue Router
- **数据可视化：** ECharts
- **HTTP请求：** Axios

## 功能模块

系统包含以下功能模块：

- 数据概览仪表盘
- 用户管理
- 角色管理
- 权限管理
- 商品管理
- 分类管理
- 系统设置
- 登录注册
- 错误页面（403/404）

## 系统角色

系统设计了基于角色的访问控制（RBAC），支持细粒度权限管理。

## 运行环境

- **推荐环境：** 现代浏览器，支持ES6+新特性
- **兼容性：** 响应式设计，完美适配PC端和移动端

## 目录结构

```
project/
│
├── public/               # 公共文件目录
├── src/
│   ├── assets/           # 静态资源目录
│   ├── components/       # 组件目录
│   ├── views/            # 页面目录
│   ├── store/            # 状态管理目录
│   ├── router/           # 路由配置目录
│   ├── utils/            # 工具方法目录
│   ├── types/            # TypeScript类型声明
│   ├── App.vue           # 主应用组件
│   └── main.ts           # 程序入口文件
│
├── .eslintrc.js          # ESLint配置文件
├── .prettierrc           # Prettier配置文件
└── vite.config.ts        # Vite配置文件
```

## 核心亮点

- **权限控制：** 完整的RBAC权限控制系统，支持细粒度权限管理
- **状态管理：** 使用Pinia进行状态管理，数据流清晰
- **数据可视化：** 采用ECharts实现丰富的数据可视化图表
- **界面设计：** Element Plus组件库，界面现代化，支持主题切换
- **代码结构：** 组件化开发，结构清晰，易于维护和扩展
- **技术规范：** 使用Vue3 Composition API，TypeScript类型支持，SCSS样式预处理，ESLint+Prettier代码规范

## 部署步骤

1. 确保环境符合要求，安装Node.js和npm
2. 克隆或下载项目代码
3. 在项目目录下运行 `npm install` 安装依赖
4. 使用 `npm run dev` 启动本地开发服务器
5. 构建生产环境代码 `npm run build`，并根据实际需求部署至服务器
